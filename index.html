Student SQL Database Lab

Use SQL queries to explore student, course, grade, attendance, and activity data. Run guided queries, edit your own SQL, and export your results as a PDF report.

Database Tables

  • Students: StudentID, FirstName, LastName, GradeLevel, Academy
  • Courses: CourseID, CourseName, Teacher, Department
  • Grades: GradeID, StudentID, CourseID, Quarter, Grade
  • Attendance: AttendanceID, StudentID, Month, DaysPresent, DaysAbsent
  • Activities: ActivityID, StudentID, ActivityName, Category

Student Challenges

1. Find all students in Grade 10.
2. Find students with grades of 90 or higher.
3. Calculate the average grade by course.
4. Identify students with more than 2 absences in a month.
5. Create a report showing student names, courses, and grades.

Teacher Tip: Students can use the guided query menu first, then modify the SQL to answer new questions.
